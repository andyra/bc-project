Modal with Folder Nav

This version reduces the number of clicks when navigating folders. Currently, you have to 1) click the select field and 2) click the folder you want. Here, we're cutting that number in half. To reduce clicks further, we could have a project pre-selected when the modal opens—perhaps the last project they copied something to.

As a bonus, you could also display the contents of the selected folder in case that extra context is helpful.

If you were moving list items, the UI would be similar, except there would be a single scrollable section of To-do lists instead of folders.
