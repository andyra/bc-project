Modal with minimal UI changes

This version would be worth checking out if the appetite for work was small since it leverages the existing workflow for moving/copying items. The main difference is that the content appears in a modal instead of a separate page. All the interaction within the modal would remain the same.

To make things even simpler, I think we could probably do away with the thumbnail since the actual thing you're moving will be visible behind the dialog backdrop.
